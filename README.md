# 🦷 DentCare - Dental Clinic Website

A modern, responsive, and user-friendly dental clinic website designed to streamline appointment booking and showcase professional dental services.

## ✨ Features
* **Modern Hero Section:** Eye-catching design highlighting patient trust and satisfaction (5,000+ Happy Patients).
* **Responsive Navigation:** Quick links to Home, Book Appointment, My Appointments, and Admin Dashboard.
* **Call to Action (CTA):** Clear buttons for "Book an Appointment" and "Contact Us" to maximize user engagement.
* **Interactive Elements:** Fully optimized for seamless navigation across desktops, tablets, and smartphones.
